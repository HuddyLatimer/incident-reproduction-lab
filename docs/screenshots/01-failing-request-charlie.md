## Why It Failed

| Field | Provided | Stored in DB | Match? |
|-------|----------|--------------|--------|
| username | `charlie` | `Charlie` | âŒ No (case mismatch) |
| password | `charlie456` | `charlie456` | âœ… Yes |

**Root Cause:** The code uses strict equality (`===`) which is case-sensitive in JavaScript.

```javascript
// BUG: This comparison is case-sensitive
const user = users.find(u => u.username === username && u.password === password);
//                                      ^^^
//                          "charlie" !== "Charlie"
```
